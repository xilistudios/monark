# Project Brief - Secure Vault Manager

## Core Objectives
- Cross-platform encrypted credential storage
- Military-grade cryptography implementation
- Intuitive desktop UI with React/Tauri
- Offline-first architecture
- Multi-language support

## Key Features
- AES-256 & ChaCha20 encryption
- Secure vault lifecycle management
- Entry grouping and tagging
- Cross-device synchronization
- Audit logging

## Technical Scope
- **Frontend**: React 19 + TypeScript
- **Backend**: Rust (Tauri) 
- **State**: Redux Toolkit
- **i18n**: JSON-based localization
