# Technical Context

## Core Stack
- **Frontend**: React 19 + Vite + TypeScript
- **Desktop**: Tauri 2.0 (Rust)
- **Styling**: TailwindCSS + Daisy UI
- **State**: Redux Toolkit
- **i18n**: i18next

## Cryptographic Foundations 
- Custom Rust implementations for:
  - Key rotation
  - Vault metadata signing
  - Secure memory management

## Dev Tooling
- Bun runtime for frontend
- Vitest for unit testing
- Tauri Android Studio integration
